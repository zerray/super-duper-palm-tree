开始界面

先选模式再开始。音效使用 Web Audio API 实时合成，首次开始会自动激活音频上下文。

玩法：看序列亮起 -> 等待“你的回合” -> 按正确顺序点击色块。答对会触发弹跳与清脆音，高亮错误会闪红并尝试触发震动。
