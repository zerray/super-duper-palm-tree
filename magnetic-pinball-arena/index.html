Physics Toy

双人磁力弹球竞技场

红蓝双方在同一块键盘上争抢一颗金属弹球。移动各自准星，放下短暂存在的磁力源，把弹球吸进或推向对方球门；随机障碍物和加速带会持续刷新，让每回合都带一点失控感。

目标 先到 5 分获胜
刷新 随机障碍物 / 加速带
磁力源 持续 3.2 秒后消散
性能 FPS 0
玩家 1 WASD + 空格

移动红色准星，按空格放置磁力源。每次放置都会在吸引 / 排斥之间轮换。

玩家 2 方向键 + Enter

移动蓝色准星，按 Enter 放置磁力源。与玩家 1 相同，磁极会轮换。

玩家 1: 0
中线开球
玩家 2: 0
R 可随时重开。把金属弹球送进对方球门即可得分。

准备开球

玩家 1 用 WASD + 空格，玩家 2 用方向键 + Enter。先到 5 分获胜。
