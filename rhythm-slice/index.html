玩法

  • 长按拖动鼠标或手指，让切割轨迹穿过形状包围盒。
  • 命中时机距离 beat 中心不超过 80ms 为 Perfect，不超过 200ms 为 Good。
  • 漏切或时机偏差超过 200ms 记为 Miss，并会打断 Perfect 连段。

反馈

粒子特效 每次切中会生成 8-12 个碎片，颜色与目标一致。
Combo 震动 连续 Perfect 达到 3 次后，后续 Perfect 会触发屏幕 shake。
内置音轨 两首 30-40 秒短曲，全部由 Web Audio API 实时合成。
结算页 歌曲结束后显示总分、各档命中统计与最高连段。
