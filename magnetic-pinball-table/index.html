玩家 1 胜利

按下 R 或点击右侧按钮立即重开。

磁力弹珠台

上下两端各有一个发射器和一枚可操控磁铁。发射弹珠去撞击对方区域得分靶，同时避开自己半场的护盾，先拿到 5 分者获胜。

玩家 1 0
玩家 2 0
目标分数 5
场上弹珠 0
状态 对战中

操作

玩家 1

W / S 调整角度，A / D 调整力度，F 发射。用鼠标左键拖动画布里的蓝色磁铁改变轨迹。

玩家 2

↑ / ↓ 调整角度，← / → 调整力度，/ 发射。使用 I / J / K / L 移动橙色磁铁。

命中对方区域的圆形得分靶可得 1 分，撞上自己区域的护盾扣 1 分。磁力场会用半透明圆环显示，弹珠保留拖尾和命中闪光。

提示：玩家 1 可以把磁铁拖到中场做拦截，玩家 2 则更适合用 IJKL 快速补位到靶前或护盾旁。
