Hex Chain Reaction

Take turns placing tokens on a hex grid. If a cell exceeds its live neighbor count, it explodes outward and can trigger a cascading capture wave.

Turn: Blue
Status: Blue to move.
Radius: 7
Move: 0

Match Controls

Two-player hot-seat. Place on empty cells or your own cells, then let the board finish every cascade before the next move.

Live State

Blue cells: 0
Red cells: 0
Cascade step: 0
Animating: No

Center cells can hold more tokens than edge cells because capacity matches the real number of neighboring hexes.

Rules

Blue and red alternate turns on the same device.
When a cell's token count exceeds its neighbor count, it bursts, sends one token to each adjacent hex, and converts those cells to the bursting player.
After the first two turns, if one player has no tokens left on the board, the other player wins immediately.

Blue Wins

Red has no tokens left on the board.
