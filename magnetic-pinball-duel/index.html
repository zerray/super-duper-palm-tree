磁力弹珠台：双人同屏对撞

一张竖版共享弹珠台，上下各有一位玩家。玩家 1 从底部往上发射，玩家 2 从顶部往下发射；弹珠会受到重力、碰撞和两枚可移动磁铁的共同影响。击中对方区域的得分靶加分，误砸己方护盾则扣分，先到目标分数的一方获胜。

玩家 1 用 W / S 调角度、A / D 调力度、F 发射；玩家 2 用 ↑ / ↓ 调角度、← / → 调力度、Enter 发射。

玩家 2 控制
↑ / ↓ 调整发射角度，← / → 调整发射力度，Enter 发射

用 I / J / K / L 微调顶部蓝色磁铁位置；触屏设备可直接拖动该磁铁。

规则
击中对方区域的得分靶 +1，击中己方护盾 -1，先到 7 分获胜

共享球台有撞柱和斜挡板，重力会持续把弹珠往下拉，磁场圆环会提示影响范围。

实现
Canvas 手写物理 + 拖尾粒子 + 磁场可视化

打开页面即可游玩，无需构建。胜利后可按 F / Enter 或点击按钮重开。
