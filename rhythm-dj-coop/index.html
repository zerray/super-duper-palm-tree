按节拍接力

左侧玩家使用 A/S/D/F，右侧玩家使用 J/K/L/;。在色块抵达左右判定线时按下对应颜色。Perfect 与 Good 会叠加共享 combo，任一侧 Miss 就会断连。

选择模式后点击开始。单人模式只生成一侧色块，方便单机试玩。

判定规则

  • Perfect: 命中误差在 50ms 内
  • Good: 命中误差在 120ms 内
  • Miss: 过线未按或按错颜色

视觉升级

共享 combo 驱动背景层级。Level 1 开始增强脉冲，Level 2 解锁色块拖尾，Level 3 加入全屏径向光晕与 Perfect 冲击。

音效实现

所有音效都由 Web Audio API 的振荡器和增益包络实时合成。每拍有底鼓，命中会按颜色切换音高，Miss 则触发一段短促失真噪声。
