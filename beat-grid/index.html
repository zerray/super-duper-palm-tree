How It Works

Tap or click highlighted pads before the timer bar empties. Wrong taps break the combo. The beat is synthesized live with Web Audio API oscillators, so there are no audio files.

Feedback

Blue glow means the pad is active.
Green flash means a successful hit.
Red flash marks a miss or bad tap.
