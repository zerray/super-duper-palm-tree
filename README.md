# Agent Workflow Fixture Repo

This repository is a minimal fixture for an agent-based workflow. It contains guidance for planner and verifier behavior alongside lightweight configuration for Claude and Codex tooling.

## Repository Structure

- `AGENTS.md`: Repository-level instructions that define scope, restrictions, and worker expectations.
- `CLAUDE.md`: Notes for planner and verifier behavior, including review and blocker-reporting style.
- `.claude/settings.json`: Claude-side model and workflow notes.
- `.codex/config.toml`: Codex-side model, approval, and sandbox configuration.
