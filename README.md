# Agent Factory

This repository contains a simple agent-factory workflow for planning and completing repository tasks with Claude and Codex agents.

## 聂小倩 HD-2D RPG

新增了一个独立原型 [`nie-xiaoqian-rpg/index.html`](/opt/agent-factory/worktrees/issue-24-r2/nie-xiaoqian-rpg/index.html)，以《聊斋志异》中的《聂小倩》为蓝本，提供：

- 郊外、兰若寺两张可探索地图
- 与燕赤霞、聂小倩的剧情对话
- 对战游魂小鬼与黑山老妖的回合制战斗

直接在浏览器打开 `nie-xiaoqian-rpg/index.html` 即可游玩。移动使用方向键或 `WASD`，交互使用空格，战斗时按 `1` 攻击、`2` 防御、`3` 法术、`4` 逃跑。
