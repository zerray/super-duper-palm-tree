# Agent Factory

This repository contains a simple agent-factory workflow for planning and completing repository tasks with Claude and Codex agents.

## 聂小倩 HD-2D RPG

新增 `nie-xiaoqian-rpg/index.html`，提供一个可直接在浏览器中打开的轻量 RPG 原型，包含：

- 郊外、兰若寺两张地图探索
- 聂小倩、燕赤霞相关剧情对话
- 小鬼伏击与黑山老妖两场回合制战斗

直接用浏览器打开 `nie-xiaoqian-rpg/index.html` 即可游玩。方向键或 `WASD` 移动，`Enter` / 空格确认或推进对话。

## Sokoban Warehouse

新增 `sokoban-game/index.html`，提供一个无需构建、可直接在浏览器中打开的 Sokoban 推箱子游戏，包含：

- 10 个关卡、步数统计、撤销与重置
- 关卡解锁与关卡选择
- `localStorage` 本地进度保存

直接用浏览器打开 `sokoban-game/index.html` 即可游玩。方向键或 `WASD` 移动，`Z` 撤销，上屏按钮适配移动端视口。
