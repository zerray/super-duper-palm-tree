# Agent Workflow Config Repo

This repository contains lightweight configuration and guidance files for an agent-based workflow. It defines planner and verifier expectations, agent working rules, and tool-specific settings for Claude and Codex.

## Repository Structure

- `AGENTS.md`: Repository-level instructions for scoped changes, restrictions, and verification expectations.
- `CLAUDE.md`: Planner and verifier guidance focused on short, executable plans and strict acceptance-criteria reviews.
- `.claude/settings.json`: Claude-specific configuration for the workflow.
- `.codex/config.toml`: Codex-specific configuration, including model and sandbox settings.
