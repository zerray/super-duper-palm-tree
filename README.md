# Agent Factory Repo

This repository contains a minimal configuration setup for an agent workflow. It defines repository-level working rules, planner/verifier guidance, and model settings for both Codex and Claude.

## Repository Structure

- `AGENTS.md`: Repository working constraints, scope limits, and change expectations.
- `CLAUDE.md`: Short guidance for planner and verifier behavior.
- `.codex/config.toml`: Codex model and execution configuration.
- `.claude/settings.json`: Claude-related model and workflow notes.

## Notes

The repository is intentionally small and focused on agent collaboration and review flow rather than application code.
